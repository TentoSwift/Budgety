# Budgety プライバシーポリシー

最終更新日: 2026年5月28日

TentoSwift（以下「当方」）は、家計簿・割り勘・精算アプリ「Budgety」（以下「本アプリ」）における利用者の情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。本アプリは iPhone / iPad / Mac / Apple Watch / Apple Vision Pro でご利用いただけます。

## 基本方針

本アプリは、利用者のデータをできる限り利用者自身の管理下に置く設計です。当方は独自のサーバーを運用しておらず、利用者の家計データ（支出・収入・シート等）を当方が収集・保管・閲覧することはありません。データは利用者の端末内、および利用者自身の iCloud アカウント（Apple の iCloud）に保存されます。

## 1. 取得・保存する情報

本アプリは、利用者が入力・操作した範囲で以下の情報を端末内に保存します。

- **家計データ**: 支出・収入の金額、日付、タイトル、メモ、カテゴリ、通貨、レシート写真、支払い者・受益者（割り勘相手）、定期項目、送金（精算）記録
- **シート情報**: シート名、色、アイコン、既定通貨、月予算、アーカイブ状態、（設定した場合）シートロック用パスワードのハッシュ値とソルト、生体認証の有効/無効設定
- **メンバー / プロフィール情報**: 表示名、アバター画像、色、Apple ID に紐づく識別子（CloudKit ユーザー識別子）、バーチャルメンバー（アプリを使っていない相手を割り勘に含める仮想プロフィール）
- **テンプレート・カテゴリ・送金（精算）記録**: 過去の送金時の為替レートも記録し、後から相場が変動しても精算結果がズレないようにするため、target 通貨換算額のスナップショットを保存します
- **アプリの設定**: 既定通貨、並び順、生体認証の有無、最後に開いたシート など

シートロックのパスワードはハッシュ化（SHA-256 + ソルト）して保存し、平文では保存しません。

## 2. データの保存場所（iCloud 同期）

- 本アプリのデータは、Apple の **CloudKit** を用いて利用者自身の **iCloud プライベートデータベース**に保存・同期されます（コンテナ識別子: `iCloud.com.tento.budgety`）。これにより、同じ Apple ID の iPhone・iPad・Mac・Apple Watch・Apple Vision Pro 間でデータが自動的に同期されます。
- iCloud 上のデータは Apple の管理下にあり、Apple のプライバシーポリシーおよび iCloud 利用条件が適用されます。当方はこれらのデータにアクセスできません。
- iCloud にサインインしていない場合、データは端末内にのみ保存されます。

## 3. プロフィールの公開（共有機能のための公開情報）

シートを他の利用者と共有する機能のため、本アプリは次の情報を **CloudKit のパブリックデータベース**に保存します。

- 表示名（displayName）
- アバター画像（プロフィール写真）

これらは、共有シート上で参加者の名前やアイコンを表示するために、**本アプリの他の利用者から参照され得る公開情報**です。公開されるのは表示名とアバターのみで、家計データ（支出・収入等）が公開されることはありません。表示名・写真を設定しない、または共有機能を利用しないことで、公開範囲を最小化できます。

## 4. シートの共有

- 共有は Apple の **CloudKit 共有（CKShare）**を利用します。
- 招待時に入力した相手のメールアドレスは、対応する iCloud（Apple ID）アカウントを検索し、招待を送るためにのみ使用されます。当方がメールアドレスを保管・二次利用することはありません。
- 共有シートのデータ（支出・カテゴリ・送金記録等）は、**オーナーが明示的に招待し、参加を承諾した参加者のみ**が閲覧・編集できます（権限はオーナーが「編集可能 / 閲覧のみ」で設定）。一般公開リンクは使用しません。
- オーナーが共有を解除した場合、または Premium が失効した場合、参加者のアクセスは取り消されます。
- 参加者がシートから退出すると、その参加者のローカルデータからは該当シートのデータが削除されます。

## 5. 外部サービスとの通信

本アプリが外部に通信するのは、為替レート取得のための次のサービスのみです。

- **frankfurter.dev**（欧州中央銀行のデータに基づく為替レート提供 API）
  - 送信する情報: **通貨コードのみ**（例: JPY、USD）。利用者の個人情報・家計データは一切送信しません。
  - 取得した為替レートは端末内にキャッシュされ、1 日 1 回程度更新されます。

## 6. AI 機能・生成機能について

本アプリの AI 機能（AI チャットによる支出分析、レシートの読み取り、カテゴリの提案、アバターの画像生成 など）は、すべて **Apple のオンデバイス AI / 生成機能**を用いて**端末内で処理**されます。

- レシートの撮影・文字認識（OCR）、解析、カテゴリ AI 提案、シート AI チャット: Apple Intelligence / Foundation Models / Vision フレームワークを使用し、**画像・テキスト・家計データが、外部サーバーや当方・第三者の AI に送信されることはありません。**
- アバターの生成: Apple の **Image Playground** を使用し、生成は端末内またはユーザーの iCloud に閉じた環境で行われます。生成された画像は本アプリのアバターとして保存されます。
- これらの機能は、対応する OS・端末（Apple Intelligence / Image Playground 対応）でのみ利用できます。

## 7. ショートカット / Siri / App Intents

本アプリは Apple の **App Intents** に対応しています。

- 「Budgety で 1200 円のラーメンを記録」のような Siri / ショートカット経由の操作は、OS 上で完結します。当方および第三者にコマンド内容や音声が送信されることはありません（音声処理は Apple の Siri が行います）。

## 8. カメラ・写真・生体認証（端末の権限）

- **カメラ**: レシートの撮影（読み取り）に使用します。
- **写真**: レシート画像やプロフィール写真の選択に使用します。
- **Face ID / Touch ID / Optic ID**: パスワードでロックしたシートを素早く開くために使用します。生体認証データは Apple の Secure Enclave で管理され、当方が取得することはありません。

これらは必要なときのみ、OS の許可ダイアログを通じて使用されます。

## 9. データのエクスポート

シートを CSV / PDF 形式でエクスポートする機能を提供しています。

- 出力先は利用者がファイル選択ダイアログで指定した場所（iCloud Drive、On My iPhone、ローカルファイル等）のみです。当方のサーバーや第三者には送信されません。
- macOS では App Sandbox の範囲内で、利用者が明示的に選んだ保存先にのみ書き込みます。

## 10. 解析・トラッキング・広告

本アプリは、**アクセス解析・行動トラッキング・広告・クラッシュレポートのための第三者 SDK を一切使用していません。** 利用者を追跡したり、個人情報を第三者と共有・販売したりすることはありません。

## 11. 課金（サブスクリプション・買い切り）について

本アプリは無料で利用でき、一部の機能を「Budgety Premium」として有料で提供します。課金は Apple の **App 内課金（StoreKit）**を通じて行われます。

### 提供する商品

- **月額サブスクリプション**（自動更新）
- **年額サブスクリプション**（自動更新）
- **ライフタイム（買い切り）**（1 回の支払いで永続利用）

価格は App Store およびアプリ内の購入画面に表示されます。

### Premium で解放される主な機能

- シートの共有（他の利用者の招待）
- シートのパスワード / 生体認証ロック
- 作成できるシート数の上限解除（無料版は自分が作成したシート 3 個まで）
- 1 シートあたりのカテゴリ数の上限解除（無料版は 20 個まで）
- バーチャルメンバー（アプリを使っていない相手を割り勘・支払者として追加）
- CSV / PDF レポート出力
- AI チャット・カテゴリ提案・Image Playground 連携などの拡張機能

無料版でも、家計簿の基本機能および iCloud 同期はご利用いただけます。

### 自動更新・支払い・管理

- サブスクリプションは、期間終了の 24 時間前までに解約しない限り自動的に更新され、Apple ID に請求されます。
- 支払いは購入確定時に Apple ID アカウントへ課金されます。
- 購入・更新・解約の管理、および購入履歴の確認は、端末の **「設定」 >（Apple ID）> サブスクリプション** から行えます。
- 「購入を復元」により、同一 Apple ID での購入を他の端末でも復元できます。
- **当方はクレジットカード番号等の決済情報を取得・保管しません。** 決済は Apple が処理し、Apple のプライバシーポリシーが適用されます。
- Premium の有効 / 無効の判定には StoreKit の購入情報を用い、その状態を端末内に保持します。

## 12. データの削除

- アプリの **「設定」> 「全データを削除」** から、シート・支出・カテゴリ・メンバー・定期項目・テンプレート・送金記録・プロフィール（名前 / 写真 / 色）・設定を含む全データを削除できます。自分が作成した共有は解除され、iCloud 上のデータからも削除されます（受信した共有はオーナー側のデータには影響しません）。
- 個別のシートは「アーカイブ」または「削除」が可能です。アーカイブはシート一覧の下部の「アーカイブ済み」セクションにまとまり、データは保持されます。削除はシートおよびその配下のすべてのデータを完全に削除します。
- アプリを削除しても iCloud 上のデータは残る場合があります。完全に削除するには、上記の「全データを削除」をご利用ください。
- 公開プロフィール（表示名・写真）は、プロフィールを変更 / 削除することで更新されます。

## 13. 子どものプライバシー

本アプリは特定の年齢層（子ども）を対象としたものではなく、子どもから意図的に個人情報を収集することはありません。

## 14. 本ポリシーの変更

本ポリシーは、必要に応じて改定することがあります。重要な変更がある場合は、アプリの更新または本ページの更新を通じてお知らせします。改定後に本アプリを利用された場合、変更に同意したものとみなします。
